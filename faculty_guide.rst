=============
Faculty Guide
=============

Faculty members have all student capabilities plus additional tools for
teaching and academic management.


Faculty-Specific Features
=========================

In addition to all student features, faculty members have access to:

- **File Upload** -- Share files with students
- **AI-assisted teaching** -- Generate quizzes, summaries, and feedback


Uploading Shared Files
======================

Faculty can upload files for students to download:

1. Click **"Files"** in the sidebar
2. Click the upload button
3. Select files and add a description
4. Files become available for all students to download


AI Chat for Teaching
====================

Faculty can use AI Chat for:

- **Generating quiz questions** from uploaded course material
- **Creating lecture summaries** using RAG with uploaded documents
- **Answering student queries** with AI-assisted responses
- **Code review** -- Paste student code for analysis and feedback
